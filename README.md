# Magna-Dashboard-
Mobile Sales Analytics Dashboard – Power BI
An interactive Power BI dashboard designed to analyze and visualize mobile sales performance across different cities, brands, payment methods, customer ratings, and time periods. This dashboard provides meaningful business insights through dynamic visualizations, helping users understand sales trends, customer behavior, and transaction patterns effectively.
Key Features
📊 Sales Performance Overview
Displays total sales, total quantity sold, total transactions, and average sales metrics.
📈 Monthly Sales Trend Analysis
Visual representation of quantity sold across different months to identify seasonal trends and performance fluctuations.
🌍 Geographical Sales Distribution
Interactive map showing sales activity across major Indian cities.
⭐ Customer Ratings Analysis
Breakdown of customer ratings to evaluate customer satisfaction levels.
💳 Payment Method Insights
Comparison of transactions made through UPI, Debit Card, Cash, and Credit Card.
📅 Day-wise Sales Analysis
Tracks sales performance based on weekdays to identify high-performing days.
📱 Brand & Mobile Model Comparison
Comparative analysis of leading smartphone brands and models based on sales and transactions
🔍 Interactive Filters
Month-wise slicer for dynamic data exploration and customized analysis.
Tools & Technologies Used
Power BI

Objective
The main objective of this project is to transform raw mobile sales data into meaningful insights that support data-driven decision-making and business intelligence analysis.
